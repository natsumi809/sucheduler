# sucheduler
自分が欲しいタスク・スケジュールアプリの作成

## 特徴
- **直感的なドラッグ＆ドロップ機能**:
  - **タスクの割り当て**: 左側の「未定タスク一覧」からカレンダーへドラッグして日付を割り当て。
  - **日付の解除**: カレンダーから「一覧に戻す」エリアへドロップして未定タスクに戻す。
  - **即時削除**: カレンダーまたは一覧から「削除エリア」へドロップしてタスクを削除。
- **動的なUI/UX**:
  - ドラッグ中のみ出現する「削除エリア」と「一覧に戻すエリア」。
  - スペースキーでのクイック入力モード。
  - マウスホイールや矢印キーによるスムーズなカレンダー月送り（フェード・スライドアニメーション付）。
- **充実のカレンダー機能**:
  - 日本の祝日をAPIから自動取得して表示。
  - タスクごとのカラーカスタマイズ。
  - 期間選択（ドラッグ選択）による複数日タスクの一括登録。
 
## 使用技術

| カテゴリ | 技術スタック |
| :--- | :--- |
| **Backend** | Python 3.x, Django |
| **Frontend** | JavaScript (ES6+), HTML5, CSS3 |
| **Libraries** | [FullCalendar 6.1.10](https://fullcalendar.io/) |
| **API** | [日本の祝日API](https://holidays-jp.github.io/) |

- 要修正機能
  + タスク一覧からタスクを削除する機能
  + タスク編集モーダルの時刻入力フォームからエンターで保存する機能
  + カレンダー上期間指定タスクの表示(現状終了日の前日までしか表示されない)

- 要変更機能
  + タスク一覧 > 「選択削除」にて、「一括削除」ボタンを「全選択」ボタンに変更

- 確認できている要修正バグ
  + カレンダーにタスクをドラッグ＆ドロップするとき勝手に期間指定になる(原因不明)
